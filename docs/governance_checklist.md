# ✅ Governance Checklist

Governance rules for academic reliability and responsible AI use.

## Dataset Governance
- [ ] class definitions frozen
- [ ] version control documented
- [ ] annotation rules applied
- [ ] dataset source recorded

## Model Governance
- [ ] metrics reproducible
- [ ] held-out test set used
- [ ] class-wise AP reported
- [ ] failure modes documented

## Deployment Governance
- [ ] human review required
- [ ] not standalone safety decision tool
- [ ] BIM mapping validated manually
